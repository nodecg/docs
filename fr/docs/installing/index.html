Installing NodeCG

Installâ

There are two methods to install NodeCG: cloning from GitHub or using the CLI.

Install Node.js (version 22).

Using the CLI:

npm install --global nodecg@latest
mkdir nodecg
cd nodecg
nodecg setup

Cloning from GitHub:

# Be sure to clone a specific release's tag name.
git clone --depth 1 --branch v2.1.0 https://github.com/nodecg/nodecg.git
cd nodecg
npm install
npm run build

Startâ

Run node index.js or nodecg start in the root nodecg folder.

Open the dashboard (http://localhost:9090 by default).

remarque

When running NodeCG in production, Docker or pm2 are recommended.

Installing bundlesâ

NodeCG's individual graphics packages are called bundles. They can be installed either from the command-line (via CLI), or by simply placing the folder into the ./bundles directory.

The easiest way to install bundles is via the command-line.

GitHubâ

nodecg install username/repo-name

BitBucketâ

nodecg install bitbucket:username/repo-name

Other Providersâ

Enter the git URL.

nodecg install https://gitlab.com/username/repo-name.git

Bundles are just directories inside the ./bundles folder. They can always be added and removed by simply moving them into or out of that folder. Avoid installing or uninstalling bundles while NodeCG is running.
